# Messagerie avec votre centre fiscal

La messagerie vous permet de contacter directement votre centre fiscal pour toute question relative à votre dossier fiscal.

## Créer une nouvelle conversation

1. Cliquez sur **Messagerie** dans la barre de navigation
2. Cliquez sur **Nouvelle conversation**
3. Saisissez un objet et votre message
4. Cliquez sur **Envoyer**

Un agent de votre centre fiscal vous répondra dans les meilleurs délais.

## Suivre vos conversations

La liste de vos conversations affiche :
- L'objet de la conversation
- La date du dernier message
- Le statut (Ouvert / Clôturé)
- Le nom de l'agent en charge (si assigné)

## Bonne pratique

Précisez toujours dans votre message :
- Votre NIF
- La période concernée
- Le type d'impôt
- La nature précise de votre question

Cela permet à l'agent de traiter votre demande plus rapidement.
