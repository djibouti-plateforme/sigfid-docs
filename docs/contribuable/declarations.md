# Mes déclarations

## Consulter mes déclarations

La section **Déclarations** liste toutes vos déclarations fiscales par période et par type d'impôt.

## Statuts

| Statut | Signification |
|--------|---------------|
| **Soumise** | Déclaration enregistrée, en attente de traitement |
| **Validée** | Traitée par votre centre fiscal |
| **Rejetée** | Des corrections sont nécessaires |

## Déclaration rejetée

Si une déclaration est rejetée, vous recevez une notification avec le motif. Contactez votre centre fiscal via la **messagerie** pour obtenir des précisions.
