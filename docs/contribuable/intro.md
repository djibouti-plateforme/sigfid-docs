# Espace contribuable SIGFID

L'espace contribuable vous permet de :
- Consulter votre **compte fiscal** et l'état de vos obligations
- Suivre vos **déclarations** en cours
- Communiquer avec votre **centre fiscal** via la messagerie sécurisée

## Accès

Votre accès est créé par votre centre fiscal lors de votre immatriculation. Vous recevez un email d'invitation pour définir votre mot de passe.

## Confidentialité

Toutes vos données fiscales sont protégées conformément au **Protocole de Confidentialité et Protection des Données SIGFID 2026**. Seuls les agents autorisés de votre centre fiscal peuvent accéder à votre dossier.
