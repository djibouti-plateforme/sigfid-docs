# Guide de l'agent des impôts

L'agent des impôts utilise **sigfid-agent** pour traiter les opérations fiscales quotidiennes : déclarations, liquidations, enregistrements et messagerie.

## Prérequis

- Un compte agent créé par un administrateur fonctionnel
- Les permissions accordées par l'administrateur pour les fonctionnalités dont vous avez besoin

:::info Permissions
Si vous ne voyez pas une fonctionnalité ou recevez un message "Accès refusé", contactez votre administrateur fonctionnel pour qu'il vous attribue la permission correspondante.
:::
