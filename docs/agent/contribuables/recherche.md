# Rechercher un contribuable

**Permission requise :** `CONTRIBUABLE_VIEW`

## Par NIF ou nom

1. Dans la barre latérale, cliquez sur **Contribuables**
2. Saisissez le NIF (Numéro d'Identification Fiscale) ou le nom dans la barre de recherche
3. Cliquez sur le contribuable dans les résultats

## Résultats de la recherche

La liste affiche pour chaque contribuable :
- NIF
- Raison sociale ou nom complet
- Centre fiscal d'affectation
- Statut (actif, suspendu, protégé)

## Contribuable protégé

Un contribuable marqué "protégé" ne peut être consulté que par les agents ayant la permission `CONTRIBUABLE_PROTECT`. Cette mesure protège les dossiers sensibles.
