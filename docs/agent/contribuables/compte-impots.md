# Compte impôts d'un contribuable

Le compte impôts centralise toutes les **dettes fiscales** d'un contribuable.

## Consulter le compte impôts

1. Ouvrez la fiche du contribuable
2. Cliquez sur l'onglet **Compte impôts**

Vous voyez :
- Les dettes en cours par type d'impôt
- Les paiements effectués
- Le solde global

## Structure d'une dette

Chaque déclaration validée génère automatiquement une ou plusieurs dettes dans le compte impôts :

| Élément | Description |
|---------|-------------|
| Type d'impôt | ITS, TVA, Patente, etc. |
| Période | Mois/année concerné |
| Montant principal | Impôt calculé |
| Pénalités | Majorations pour retard |
| Statut | En attente, partiel, soldé |

## Enregistrer un paiement

1. Dans le compte impôts, cliquez sur **Enregistrer un paiement**
2. Saisissez le montant et le mode de paiement
3. Confirmez

Le solde est mis à jour immédiatement.
