# Se connecter à sigfid-agent

1. Ouvrez sigfid-agent dans votre navigateur
2. Cliquez sur **Se connecter**
3. Saisissez votre email et mot de passe

## Première connexion

Vous recevez un email d'invitation lors de la création de votre compte. Cliquez sur le lien pour définir votre mot de passe (valable 72 heures).

## Mode hors ligne

SIGFID fonctionne en **mode hors ligne** : si votre connexion internet est interrompue, vous pouvez continuer à consulter les données chargées. Les actions de modification sont mises en file d'attente et envoyées automatiquement à la reconnexion.

Un indicateur dans l'en-tête de l'application affiche votre état de connexion.
