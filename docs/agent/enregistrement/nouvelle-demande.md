# Créer une demande d'enregistrement

**Permission requise :** `ENREGISTREMENT_CREATE`

Les droits d'enregistrement s'appliquent aux actes juridiques (contrats, cessions, mutations de propriété).  
**Référence CGI :** Livre 2, Partie 1.

## Étapes

1. Dans la barre latérale, cliquez sur **Enregistrement → Nouvelle demande**
2. Sélectionnez le **type d'acte** (vente immobilière, cession de fonds de commerce, bail, etc.)
3. Renseignez les parties (vendeur, acheteur, ou parties au contrat)
4. Saisissez la **valeur vénale** ou le **montant du loyer** selon l'acte
5. Joignez les documents requis (acte notarié, bail signé) via le service document
6. Cliquez sur **Soumettre la demande**

## Documents requis

Selon le type d'acte :
- Copie de l'acte notarié ou sous seing privé
- Titre de propriété
- Pièces d'identité des parties
