# Valider une demande d'enregistrement

**Permission requise :** `ENREGISTREMENT_VALIDATE`

## Consulter les demandes en attente

1. Cliquez sur **Enregistrement → Demandes en attente**
2. La liste affiche toutes les demandes soumises en attente de validation

## Valider une demande

1. Ouvrez la demande
2. Vérifiez les informations et les documents joints
3. SIGFID calcule automatiquement les droits dus selon le type d'acte et la valeur déclarée
4. Si tout est correct, cliquez sur **Valider**
5. Une dette est créée dans le compte impôts du contribuable

## Rejeter une demande

Si les informations sont incomplètes ou incorrectes :
1. Cliquez sur **Rejeter**
2. Saisissez le motif du rejet
3. Le contribuable est notifié via la messagerie

## Calcul des droits d'enregistrement

Les droits sont calculés selon le barème CGI (Livre 2, Partie 1) en fonction du type d'acte et de la valeur. SIGFID applique automatiquement le bon taux depuis les paramètres fiscaux.
