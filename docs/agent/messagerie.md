# Messagerie avec les contribuables

**Permission requise :** `REPONDRE_MESSAGERIE`

La messagerie permet aux agents de communiquer directement avec les contribuables via la plateforme SIGFID.

## Consulter les conversations

1. Dans la barre latérale, cliquez sur **Messagerie**
2. La liste affiche toutes les conversations de votre centre fiscal

Chaque conversation indique :
- Le sujet
- Le contribuable
- La date du dernier message
- Le statut (Ouvert / Clôturé)

## Répondre à un message

1. Ouvrez la conversation
2. Saisissez votre réponse dans la zone de texte
3. Cliquez sur **Envoyer**

Le contribuable reçoit une notification et peut consulter votre réponse dans son espace.

## Clôturer une conversation

Lorsque la question du contribuable est résolue, cliquez sur **Clôturer la conversation**. Le contribuable peut toujours consulter l'historique mais ne peut plus envoyer de nouveaux messages dans cette conversation.
