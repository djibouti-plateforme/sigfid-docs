# Gérer les agents des impôts

## Créer un nouvel agent

1. Dans la barre latérale, cliquez sur **Agents**
2. Cliquez sur **Nouvel agent**
3. Remplissez le formulaire :
   - Prénom et nom (obligatoires)
   - Adresse email (obligatoire, doit être unique)
   - Téléphone (optionnel)
   - Centre fiscal d'affectation
4. Cliquez sur **Créer**

L'agent reçoit automatiquement un email d'invitation avec un lien pour définir son mot de passe.

## Statuts d'un agent

| Statut | Signification |
|--------|---------------|
| **En attente** | Invitation envoyée, mot de passe non encore défini |
| **Actif** | Compte activé, agent peut se connecter |
| **Suspendu** | Accès temporairement bloqué |

## Suspendre / Réactiver un agent

1. Ouvrez la fiche de l'agent
2. Dans la section **Actions**, cliquez sur **Suspendre** ou **Réactiver**

Un agent suspendu ne peut plus se connecter à sigfid-agent. Ses données et historiques sont conservés.

## Transférer un agent vers un autre centre fiscal

1. Ouvrez la fiche de l'agent
2. Cliquez sur **Transférer**
3. Sélectionnez le nouveau centre fiscal
4. Renseignez le motif du transfert (optionnel)
5. Confirmez

L'historique des transferts est conservé et consultable dans la fiche de l'agent.

## Supprimer un agent

:::danger
La suppression est irréversible. Préférez la suspension si vous souhaitez conserver l'historique.
:::

1. Ouvrez la fiche de l'agent
2. Cliquez sur **Supprimer**
3. Confirmez la suppression dans la boîte de dialogue
