# Gérer les administrateurs fonctionnels

## Créer un administrateur

1. Dans la barre latérale, cliquez sur **Administrateurs**
2. Cliquez sur **Nouvel administrateur**
3. Remplissez le formulaire (prénom, nom, email)
4. Cliquez sur **Créer**

L'administrateur reçoit un email d'invitation pour définir son mot de passe.

:::info
Il doit exister au moins un administrateur actif à tout moment dans le système. Vous ne pouvez pas supprimer votre propre compte.
:::

## Suspendre / Réactiver

Même procédure que pour les agents. Un administrateur suspendu ne peut plus accéder à sigfid-console.
