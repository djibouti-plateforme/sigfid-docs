# Se connecter à sigfid-console

## Prérequis

- Un compte administrateur fonctionnel créé par un autre administrateur
- L'URL de l'application sigfid-console fournie par votre responsable informatique

## Étapes de connexion

1. Ouvrez sigfid-console dans votre navigateur
2. Cliquez sur **Se connecter**
3. Saisissez votre adresse email et votre mot de passe
4. Vous êtes redirigé vers le tableau de bord

## Première connexion

Lors de votre première connexion, vous recevez un email d'invitation contenant un lien pour définir votre mot de passe. Ce lien est valable **72 heures**.

:::warning
Si le lien a expiré, demandez à un autre administrateur de vous renvoyer une invitation depuis **Administrateurs → Détail → Renvoyer l'invitation**.
:::

## Déconnexion

Cliquez sur votre nom en haut à droite puis **Déconnexion**. La session expire automatiquement après 8 heures d'inactivité.
