# Gérer les centres fiscaux

Les centres fiscaux sont les entités administratives auxquelles sont affectés les agents et les contribuables.

## Créer un centre fiscal

1. Dans la barre latérale, cliquez sur **Centres fiscaux**
2. Cliquez sur **Nouveau centre**
3. Remplissez :
   - **Code** : identifiant unique (ex : `CF-DJI-001`)
   - **Nom** : nom complet du centre
   - **Région** : région administrative de Djibouti
4. Cliquez sur **Créer**

## Régions disponibles

- Djibouti
- Ali Sabieh
- Dikhil
- Tadjourah
- Obock
- Arta

## Modifier un centre fiscal

Ouvrez la fiche du centre et cliquez sur **Modifier**. Vous pouvez mettre à jour le nom et la région. Le code est immuable après création.
