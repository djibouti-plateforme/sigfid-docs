# Functional Administrator Guide

The functional administrator is responsible for the **configuration and parameterisation** of the SIGFID system. They do not carry out any tax operation — that role is reserved for tax officers.

## Responsibilities

| Domain | Actions |
|--------|---------|
| **Users** | Create and manage officer and administrator accounts |
| **Permissions** | Grant and revoke access rights to officers |
| **Tax centres** | Create and configure tax centres |
| **Tax parameters** | Enter rates, brackets and tax thresholds |

## Fundamental principle

> The functional administrator **configures** the system. The tax officer **uses** it to process tax cases.

Tax parameters (tax rates, brackets, thresholds) are **always entered in the database** by the administrator — never hard-coded in the system. If the Finance Act amends a rate, only the database is updated.

## Application access

The functional administrator accesses **sigfid-console**, the dedicated administration interface.
