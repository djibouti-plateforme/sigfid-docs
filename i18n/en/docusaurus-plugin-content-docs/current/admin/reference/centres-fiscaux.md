# Manage tax centres

Tax centres are the administrative entities to which officers and taxpayers are assigned.

## Create a tax centre

1. In the sidebar, click **Tax centres**
2. Click **New centre**
3. Fill in:
   - **Code**: unique identifier (e.g. `CF-DJI-001`)
   - **Name**: full name of the centre
   - **Region**: administrative region of Djibouti
4. Click **Create**

## Available regions

- Djibouti
- Ali Sabieh
- Dikhil
- Tadjourah
- Obock
- Arta

## Modify a tax centre

Open the centre's profile and click **Edit**. You can update the name and the region. The code is immutable after creation.
