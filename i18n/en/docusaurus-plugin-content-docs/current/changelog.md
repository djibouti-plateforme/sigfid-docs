# Changelog

This page lists major changes to the SIGFID documentation. Minor changes (spelling fixes, formatting) are not listed here — see the [Git history](https://github.com/boubacarDiatta/sigfid-docs/commits/main) for the complete log.

:::info Legal reference
A change in this documentation reflects a change in tax law **only if it explicitly cites a CGI article or LFI**. All other changes are editorial (clarification, restructuring, examples).
:::

---

## 2026

### 🌍 English version — *in progress*

- Bilingual French / English support
- All pages translated into English
- Locale dropdown in the navbar
- English UI strings (theme, navbar, footer)

### 🎨 Design and security overhaul

- Institutional graphic identity (palette inspired by the Djibouti flag)
- DGI logo, favicon, and dedicated Open Graph social card
- Redesigned homepage with persona journeys and key figures
- Mermaid diagrams replacing ASCII workflows
- Cross-cutting pages: glossary, FAQ, changelog
- Repository security: `LICENSE`, `SECURITY.md`, Dependabot, CodeQL, automatic link checking, markdown linting
- Hardened deployment workflow (timeouts, minimal permissions, npm cache)

---

## 2025

### 📚 Initial publication of SIGFID documentation

- Complete documentation of tax modules: ITS, VAT, IBP/IMF, Patentes, Licences, IRNR, RAS PS, other taxes (IIS, IPVI, IPVM, IRVM, TBA)
- Distinct guides for the three user profiles: functional administrator, tax officer, taxpayer
- Numerical scenarios on each tax
- Systematic referencing of applicable CGI articles and LFIs
- Set-up of the GitHub Pages deployment pipeline
- Adoption of Docusaurus 3.10 as documentation engine
