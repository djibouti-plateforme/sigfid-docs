# My returns

## View my returns

The **Returns** section lists all your tax returns by period and by tax type.

## Statuses

| Status | Meaning |
|--------|---------|
| **Submitted** | Return registered, awaiting processing |
| **Validated** | Processed by your tax centre |
| **Rejected** | Corrections are required |

## Rejected return

If a return is rejected, you receive a notification with the reason. Contact your tax centre via **messaging** for further details.
