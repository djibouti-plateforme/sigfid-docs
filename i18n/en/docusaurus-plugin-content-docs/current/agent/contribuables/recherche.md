# Search a taxpayer

**Required permission:** `CONTRIBUABLE_VIEW`

## By NIF or name

1. In the sidebar, click **Taxpayers**
2. Enter the NIF (Tax Identification Number) or the name in the search bar
3. Click on the taxpayer in the results

## Search results

The list displays for each taxpayer:
- NIF
- Business name or full name
- Assigned tax centre
- Status (active, suspended, protected)

## Protected taxpayer

A taxpayer marked as "protected" can only be viewed by officers holding the `CONTRIBUABLE_PROTECT` permission. This measure protects sensitive files.
