# Taxpayer's tax account

The tax account centralises all the **tax debts** of a taxpayer.

## View the tax account

1. Open the taxpayer's profile
2. Click the **Tax account** tab

You see:
- Outstanding debts by tax type
- Payments made
- Overall balance

## Structure of a debt

Each validated return automatically generates one or several debts in the tax account:

| Element | Description |
|---------|-------------|
| Tax type | ITS, VAT, Patente, etc. |
| Period | Month/year concerned |
| Principal amount | Calculated tax |
| Penalties | Surcharges for delay |
| Status | Pending, partial, settled |

## Record a payment

1. In the tax account, click **Record a payment**
2. Enter the amount and the payment method
3. Confirm

The balance is updated immediately.
