# Tax Officer Guide

The tax officer uses **sigfid-agent** to process daily tax operations: returns, assessments, registrations and messaging.

## Prerequisites

- An officer account created by a functional administrator
- Permissions granted by the administrator for the features you need

:::info Permissions
If you do not see a feature or receive an "Access denied" message, contact your functional administrator to have the corresponding permission granted.
:::
