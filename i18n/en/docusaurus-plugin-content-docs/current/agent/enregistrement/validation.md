# Validate a registration request

**Required permission:** `ENREGISTREMENT_VALIDATE`

## View pending requests

1. Click **Registration → Pending requests**
2. The list shows all submitted requests awaiting validation

## Validate a request

1. Open the request
2. Check the information and attached documents
3. SIGFID automatically calculates the duties owed according to the type of act and the declared value
4. If everything is correct, click **Validate**
5. A debt is created in the taxpayer's tax account

## Reject a request

If the information is incomplete or incorrect:
1. Click **Reject**
2. Enter the reason for rejection
3. The taxpayer is notified via messaging

## Calculation of registration duties

Duties are calculated according to the CGI scale (Book 2, Part 1) based on the type of act and the value. SIGFID automatically applies the correct rate from the tax parameters.
