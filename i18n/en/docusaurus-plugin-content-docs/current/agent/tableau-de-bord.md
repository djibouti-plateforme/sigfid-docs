# Officer dashboard

The sigfid-agent dashboard provides a synthetic overview of the tax centre's activity.

## Indicators displayed

### Today's activity
- Number of returns entered today
- Number of returns validated
- Total amount of taxes assessed

### Alerts and deadlines
- Active taxpayers without VAT return (deadline D−3, D−1, overdue)
- Active taxpayers without ITS return
- Notices of Tax Collection (AMR) awaiting collection
- Unread messaging messages

### Tax centre statistics
- Total revenue for the current month by tax type
- Comparison with the previous month
- Filing compliance rate

## Quick actions

From the dashboard, the officer can directly:
- **New return**: quick access by tax type
- **Search a taxpayer**: by NIF or name
- **View messages**: messaging with taxpayers

## Offline mode

When the internet connection is interrupted, the dashboard displays cached data with the label **"Offline mode — data from [date]"**. Actions are queued and synchronised upon reconnection.
