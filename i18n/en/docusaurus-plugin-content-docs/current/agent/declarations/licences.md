# Licence Contribution (CL)

**CGI reference:** Book 1, Title 2, Chapter 3 — Art. 130 to 136  
**Required permission:** `DECLARATION_LICENCE_CREATE`

---

## 1. Scope (Art. 130)

Any natural or legal person having obtained the **administrative authorisation** to engage in the beverage trade, including any mess, club, restaurant, hotel.

:::info CL / Patente articulation
A beverage seller pays **BOTH**:
- The **patente** for the commercial activity
- The **licence** for the authorisation to sell beverages
SIGFID handles this dual obligation and blocks validation if one of the two is missing.
:::

---

## 2. Annual character (Art. 131)

The licence is owed for the **full year**, regardless of the opening date. No pro rata temporis (unlike the patente).

---

## 3. Schedule — 8 licence classes (Art. 132, Annex Art. 136)

### Category I — On-premise consumption

| Class | Type of establishment | Annual amount |
|-------|----------------------|---------------|
| 1st class A | Luxury restaurants and hotels | Highest rate |
| 1st class B | 1st-category restaurants and hotels | — |
| 2nd class A | Night bars, nightclubs | — |
| 2nd class B | Cafés, brasseries | — |
| 3rd class | Station buffets, snack bars | — |
| 4th class | Approved sports clubs and associations | — |

### Category II — Off-premise sale

| Class | Type of establishment |
|-------|----------------------|
| 7th class | Temporary licence (parties, events) |
| 8th class | Wholesalers |

*Exact amounts are configured in SIGFID tax parameters by the functional administrator.*

---

## 4. Enter a licence return in SIGFID

1. Click **Returns → Licences → New return**
2. Select the taxpayer
3. Select the **licence class** (1st A, 2nd B, etc.)
4. SIGFID automatically retrieves the **rate** from the tax parameters
5. Validate

---

## 5. Specific penalties (Art. 257-259)

| Infringement | Penalty | Article |
|--------------|---------|---------|
| Sale without licence | Fine + administrative closure | Art. 257 |
| Failure to display the licence | Fine | Art. 259 |
